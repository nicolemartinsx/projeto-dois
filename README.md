# Projeto Dois

Instalar o Herd  
https://herd.laravel.com/

Ter um servidor MySQL

Instruções de instalação:
```
composer install
npm install
php artisan migrate:fresh
php artisan db:seed
herd tld com
npm run dev
```

Adicionar no arquivo `hosts` o domínio do projeto:
```
127.0.0.1 projeto-dois.com
```
